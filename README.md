# Dotfiles

My config files for:

- Zsh
- Tmux
- Alacritty
- NeoVim
- Helix
- LazyGit
- Yabai
- Skhd

## Installation

1. Git clone this into home directory
2. `cd dotfiles`
3. Run `./scripts/install-[YOUR_PACKAGE_MANAGER]`
4. Run `stow .`
5. Run `./scripts/install-tpm`
6. Open tmux and press `C-b I` to install plugins
